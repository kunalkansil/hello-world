Git commands on cmd:

git clone URL: take a repository that is somewhere remotely on to your local machine. clone a repository from net to your machine.

git status: shows the status of the open branch('master' by default); branch-name, untracked files, umcommited changes

git checkout {branch-name(gh-pages)}: open the branch

git add. : to add untracked files(changes made) to git; marked to be committed

git commit: to commit the files in local repository

git commit -m "message": message is displayed with the commit

git push: to commit the files to the remote(github) repository
